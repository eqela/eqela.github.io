class cape.Array

The Array class provides convenience methods for dealing with static array objects. For dynamic array support, use vectors instead.

Static Methods
func asObject ( array as array<T> ) as cape.ArrayObject<T>
Returns the given array as an ArrayObject (which is an object type) that can be used wherever an object is required.
func isEmpty ( array as array<T> ) as bool
func isNotEmpty ( array as array<T> ) as bool
func toVector ( array as array<T> ) as vector<T>
Creates a new vector object that will be composed of all the elements in the given array. Essentially converts an array into a vector.
