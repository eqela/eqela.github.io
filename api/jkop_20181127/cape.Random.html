class cape.Random

The Random class provides means to generate random number of different kinds.

Static Methods
func getSharedInstance as cape.Random
Gets a common, shared instance of a random number generator.
Instance Methods
func setSeed ( seed as int64 ) as void
Initializes the random number generator with the given seed, if applicable. Note that the exact behavior is platform dependent: Not all platforms require nor utilize a seed at all.
func nextBoolean as bool
Used to retrieve a random boolean value (true or false)
func nextInt as int32
Used to retrieve random integer values between -2147483647 (inclusive) and 2147483648 (exclusive).
func nextInt ( n as int32 ) as int32
Used to retrieve integer values from 0 (inclusive) and "n" (exclusive)
func nextInt ( s as int32 , e as int32 ) as int32
Used to retrieve integer values between "s" (inclusive) and "e" (exclusive)
func nextBytes ( buf as array<int8> ) as void
Used to retrieve an array of random bytes.
func nextDouble as double
Used to retrieve a random double-precision floating point number with value between 0.0 (inclusive) and 1.0 (exclusive).
func nextFloat as float
Used to retrieve a random single-precision floating point number with value between 0.0 (inclusive) and 1.0 (exclusive).
