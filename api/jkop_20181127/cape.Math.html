func round ( n as double ) as double
Rounds the given number to the nearest integer. For halfway cases where the difference from both possible values is exactly 0.5, this function uses the default mechanism of the underlying platform. This means that the exact behavior of halways cases varies between platforms. To have complete control over this, use the other round() method and specify the desired halfway behavior using the "mode" parameter.
func round ( d as double , mode as int ) as double
Rounds the given number to the nearest integer. For halway cases where the difference from both possible values is exactly 0.5, this function can behave in different ways, depending on the "mode" parameter that is given (see rounding mode constants).
