class jk.text.StringUtil

The StringUtil class provides commonly used generic utility functions for string manipulation.

Static Methods
func padToLength ( value as string , length as int , align as int , paddingCharacter as char ) static as string
Creates a new string based on "string" that is at least "length" characters long. If the original string is shorter, it will be padded to the desired length by adding instances of paddingCharacter (default of which is the space character). The "aling" attribute may be either -1, 0 or 1, and determines if the padding characters willb e added to the end, both sides or to the beginning, respectively.
func quotedStringToVector ( str as string , delim as char , ignoreRepeatedDelimiters as bool , escapeCharacter as char ) static as vector<string>
Splits the given string "str" to a vector of multiple strings, cutting the string at each instance of the "delim" character where the delim does not appear enclosed in either double quotes or single quotes. Eg. quotedStringToVector("first 'second third'", ' ') produces [ "first", "second third" ]
func quotedStringToMap ( str as string , delim as char ) static as map<string,string>
Parses the string "str", splitting it to multiple strings using quotedStringToVector(), and then further processing it to key/value pairs, splitting each string at the equal sign '=' and adding the entries to a map.
func combine ( strings as vector<string> , delim as char , unique as bool , index as int ) static as string
Combines a vector of strings to a single string, incorporating the "delim" character between each string in the vector. If the "unique" variable is set to "true", only one instance of each unique string will be appended to the resulting string.
func validateCharacters ( str as string , validator as function<bool,char> ) static as bool
Vaidates a string character-by-character, calling the supplied function to determine the validity of each character. Returns true if the entire string was validated, false if not.
func toStringVector ( vector as vector , allowConversion as bool ) static as vector<string>
Converts a vector of objects to a vector of strings. If the allowConversion parameter is true, attempts to also convert any non-string objects in the original vector to string objects, if possible. In either case, any objects that are either not strings or not convertible to strings, will be omitted from the resulting vector.
func toObjectVector ( vector as vector<string> ) static as vector
Converts a vector of strings to a vector of objects. No operations are performed on the elements.
func wrapToLines ( text as string , charactersPerLine as int ) static as vector<string>
Wraps the given strings to a vector of multiple lines, honoring a maximum of charactersPerLine characters on each line. Attempts to break lines at whitespace where possible.
func stripLeadingZeros ( str as string ) static as string
Removes all "0" characters from the beginning of the string.
func formatNumberString ( vv as string , delimiter as char ) static as string
Takes a string that represents a number, and presents it in a nicely formatted manner, with delimiter separating each sequence of three digits.
func formatInteger ( vv as int , delimiter as char ) static as string
Converts an integer value to a string and presents it in a nicely formatted manner, with delimiter separating each sequence of three digits.
func formatLongInteger ( vv as long , delimiter as char ) static as string
Converts a long integer value to a string and presents it in a nicely formatted manner, with delimiter separating each sequence of three digits.
