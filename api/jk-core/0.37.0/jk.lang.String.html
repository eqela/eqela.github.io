class jk.lang.String

The String class provides all the common string manipulation functions, including comparisons, concatenation, transformations to upper and lowercase, getting substrings, finding characters or substrings, splitting the strings, converting to byte buffers, etc.

The default padding character is "0", but can be customized with the "paddingString" parameter. eg. String.forIntegerWithPadding(9, 3, "0") would yield "009". eg. Compares two strings, and returns an integer value representing their sorting order. The return value 0 indicates that the two strings are equivalent. A negative return value (less than 0) indicates that "str1" is less than "str2", and a positive return value (greater than 0) indicates that "str1" is greater than "str2". If either "str1" or "str2" is null, the comparison yields the value 0.
