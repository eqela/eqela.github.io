interface jk.app.ApplicationContext

An ApplicationContext represents the environment in which an application executes. The implementation of this is system dependent, and depending on the actual environment, the object that implements ApplicationContext may also implement other interfaces that provide additional features into that particular environment.

Base Types
implements jk.log.LoggingContext
Instance Methods
func getApplicationDataDirectory as jk.fs.File
Returns a File object that represents a directory where this application can store private data. The data is specific to the current applications and (if applicable) the current user. The exact location and implementation fully depends on the underlying operating system.
