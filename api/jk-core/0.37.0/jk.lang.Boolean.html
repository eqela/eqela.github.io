class jk.lang.Boolean

The Boolean class provides convenience methods for dealing with boolean values (either true or false).

Base Types
implements jk.lang.BooleanObject
Static Methods
func asObject ( value as bool ) static as jk.lang.Boolean
Returns the given boolean value as a BooleanObject (which is an object type) that can be used wherever an object is required.
func asBoolean ( obj as dynamic , defaultValue as bool ) static as bool
Converts the given object to a boolean value, as much as is possible. Various different objects can be supplied (including instances of BooleanObject, IntegerObject, StringObject, DoubleObject, CharacterObject, ObjectWithSize as well as strings.
