class jk.time.SystemClock

Static Methods
func asDateTimeUTC static as jk.lang.DateTime
func asDateTimeLocal static as jk.lang.DateTime
func getTimezoneOffsetAsSeconds static as long
Returns the number of seconds of timezone difference in the current system. The value is expressed as the number of seconds to ADD the the local time in order to get the UTC time equivalent.
func asUTCSeconds static as long
Returns the number of seconds since January 1, 1970, in UTC timezone.
func asSeconds static as long
Returns the number of seconds since January 1, 1970, in local timezone.
func asTimeValue static as jk.lang.TimeValue
Returns the current system time as a TimeValue structure (up to microsecond precision). No guarantee is made as to the timezone (whether local time or UTC). Therefore, TimeValue structures created this way are mostly usable for comparing the difference between two objects retrieved at different times.
func update ( v as jk.lang.TimeValue ) static as void
Updates an existing TimeValue object with the current time. This way you can reuse the object and do not need to allocate new objects, which is especially beneficial when updating the TimeValue frequently.
