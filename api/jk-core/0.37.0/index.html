JK Core [0.37.0] - API Reference

eqela:jk-core:0.37.0

Core fundamental framework for software application development, including features for data structures, string manipulation, filesystem access, I/O, threading and other commonly used operating system features.

jk.app - Application context

Application abstraction that can be used in the implementation of different kinds of application types.

interface ApplicationContext
An ApplicationContext represents the environment in which an application executes. The implementation of this is system dependent, and depending on the actual environment, the object that implements ApplicationContext may also implement other interfaces that provide additional features into that particular environment.
