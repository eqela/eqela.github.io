JK DB [0.21.0] - API Reference

eqela:jk-db:0.21.0

jk.appdb - Application database

A convenient utility for declaring and maintaining the database structure and connection for an application.

class ApplicationDatabase
Base class for application-specific database schema declarations.

jk.mssql - Microsoft SQL Server support

Client support for Microsoft SQL Server products.

class MSSQLDatabase
Microsoft SQL Server database driver.

jk.mysql - MySQL support

Client support for MySQL databases.

class MySQLDatabase
MySQL database driver.
class MySQLDatabaseGeneric
A generic MySQL client implementation, fully written in Sling. This class is not normally meant to be used directly but through MySQLDatabase.

jk.sql - SQL database access framework

Base classes and APIs for all SQL support for JK applications.

class SQLDatabase
SQL database connection. This is the base class and abstraction for all database-specific driver classes.
class SQLOrderingRule
Representation of an ordering rule in SQL.
class SQLResultSetIterator
A generic iterator for SQL result sets.
class SQLResultSetSingleColumnIterator
A generic iterator that iterater a single column in a result set.
interface SQLStatement
Represents an SQL statement or query
class SQLTableColumnIndexInfo
Information about column indexes of an SQL table.
class SQLTableColumnInfo
Information about columns of an SQL table.
class SQLTableInfo
Information about an SQL table.

jk.sql.factory - SQL database factory

Create database connections and assign drivers dynamically.

class SQLDatabaseFactory
A generic utility class for establishing connections to any supported database based on a connection string.

jk.sqlite - SQLite support

Client support for SQLite databases.

class SQLiteDatabase
SQLite database driver.
