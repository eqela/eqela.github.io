class jk.sql.factory.SQLDatabaseFactory

A generic utility class for establishing connections to any supported database based on a connection string.

Static Methods
func getDatabase ( ctx as jk.log.LoggingContext , connectionString as string ) static as jk.sql.SQLDatabase
Allocates a driver and connects to the database as specified by the connectionString. Supported connection strings start with "mssql:", "mysql:" or "sqlite:".
func getDatabaseWithConfig ( ctx as jk.log.LoggingContext , config as jk.lang.DynamicMap ) static as jk.sql.SQLDatabase
Allocates a driver and connects to the database as specified by the config map. The "type" property in the config map specifies the type of the database. Currently only "mysql" is supported.
